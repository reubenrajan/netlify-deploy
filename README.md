# AI App Playgound
Deployment Repository for AI Apps deployed to Netlify

## [Resilience Test Engineering Interview Questionnaire](https://projects.reubenrajan.dev/resilience-interview/)
A self-contained interview toolkit for hiring a Resilience Test Engineer, built around a research CSV that catalogues data points across 9 resilience areas (e.g., reliability targets, resilient architecture, disaster recovery, chaos testing, and monitoring maturity) with references to Azure, AWS, and industry documentation. A detailed prompt file [prompt.md](./resilience-interview/prompt.md) specifies how to generate an interactive, single-file HTML questionnaire featuring 90 scored questions with answer hints, checkboxes, session save/load, dark/light theming, and a live scoring widget with hire/no-hire verdicts. The resulting [questionnaire.html](https://questionnaire.reubenrajan.dev/) is the fully built, production-grade interactive interview tool delivered as plain HTML, CSS, and vanilla JavaScript with no external dependencies beyond Google Fonts.

## [Bible Art Attack](https://projects.reubenrajan.dev/bible-art-attack/)
A detailed design document for building "Bible Art Attack!! 🎨", a polished interactive web game for church/youth events where contestants generate AI images of Bible events in random art styles. The blueprint outlines a modern, playful, and visually engaging single-page web application with inline CSS and JavaScript, featuring a fun game for game night.

## Resilience Test Engineering Interview Questionnaire
[Resilience Test Engineering Interview Questionnaire (Netlify)](https://resilience-test-engineer-interview.netlify.app/)

The resilience-interview directory is a self-contained interview toolkit for hiring a Resilience Test Engineer, built around a research CSV that catalogues data points across 9 resilience areas (e.g., reliability targets, resilient architecture, disaster recovery, chaos testing, and monitoring maturity) with references to Azure, AWS, and industry documentation. A detailed prompt file [prompt.md](./resilience-interview/prompt.md) specifies how to generate an interactive, single-file HTML questionnaire featuring 90 scored questions with answer hints, checkboxes, session save/load, dark/light theming, and a live scoring widget with hire/no-hire verdicts. The resulting [questionnaire.html](https://resilience-test-engineer-interview.netlify.app/) is the fully built, production-grade interactive interview tool delivered as plain HTML, CSS, and vanilla JavaScript with no external dependencies beyond Google Fonts.
